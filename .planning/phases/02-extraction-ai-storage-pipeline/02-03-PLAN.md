**1. Create `lib/memsync.ts`** -- MemSync REST API client:

```typescript
export class MemSyncClient {
  private baseUrl = 'https://api.memchat.io/v1';
  constructor(private apiKey: string) {}

  async saveMemories(tasks: Task[], threadId: string): Promise<{ success: boolean; error?: string }>
  async searchMemories(query: string, limit?: number): Promise<{ memories: unknown[]; error?: string }>
}
```

- `saveMemories`: POST to `/v1/memories` with messages array (each task formatted as a message with role: 'user', content: structured text describing the task), agent_id: 'opengradient-task-assistant', thread_id, source: 'browser-extension'. Headers: Content-Type application/json, X-API-Key.
- `searchMemories`: POST to `/v1/memories/search` with query, limit (default 10), rerank: true. Same auth headers.
- Both methods use try/catch and return `{ success: false, error: ... }` on failure, never throw.
- Import Task type from `@/lib/types`.

**2. Create `lib/storage.ts`** -- Offline-first dual-write storage layer:

Export these functions:

```typescript
export async function saveTasksLocally(tasks: Task[]): Promise<void>
```
- Reads existing tasks from `chrome.storage.local` (key: 'tasks', defaults to [])
- Deduplicates by task.id (skip tasks whose ID already exists)
- Writes merged array back to chrome.storage.local
- Does NOT call MemSync -- that is handled separately

```typescript
export async function getLocalTasks(): Promise<Task[]>
```
- Reads tasks from chrome.storage.local (key: 'tasks', defaults to [])
- Returns them sorted by createdAt descending (newest first)

```typescript
export async function syncToMemSync(tasks: Task[]): Promise<void>
```
- Reads MemSync API key from chrome.storage.local (key: 'memsyncApiKey')
- If no API key, log a warning and return silently (MemSync is optional for MVP)
- Creates MemSyncClient instance
- Calls saveMemories with the tasks array and a thread_id derived from the current date (e.g., `tasks-2026-03-01`)
- On success, marks tasks as synced: updates each task's `synced` field to `true` in chrome.storage.local
- On failure, calls `markForRetry` and logs a warning

```typescript
export async function markForRetry(tasks: Task[]): Promise<void>
```
- Reads pending retry queue from chrome.storage.local (key: 'memsyncRetryQueue', defaults to [])
- Adds task IDs to the queue (deduped)
- Writes back to chrome.storage.local
- This is a simple "mark for later" -- actual retry logic is deferred to Phase 3+

Import Task type from `@/lib/types`, MemSyncClient from `@/lib/memsync`.

**TRIGGER_EXTRACTION handler:**
When the popup sends `{ type: 'TRIGGER_EXTRACTION', platform: 'telegram' | 'selection' }`:

The full flow in the TRIGGER_EXTRACTION handler:

```
async () => {
  // 1. Get active tab
  const [tab] = await chrome.tabs.query({ active: true, currentWindow: true });
  if (!tab?.id) { sendResponse({ success: false, error: 'No active tab' }); return; }

  // 2. Ask content script to extract text
  const extraction = await browser.tabs.sendMessage(tab.id, {
    type: 'TRIGGER_EXTRACTION',
    platform: message.platform ?? 'selection',
  });

  if (!extraction?.text?.trim()) {
    sendResponse({ success: false, error: 'No text found. Select text or open a Telegram chat.' });
    return;
  }

  // 3. Get private key
  const { ogPrivateKey } = await chrome.storage.session.get('ogPrivateKey');
  if (!ogPrivateKey) {
    sendResponse({ success: false, error: 'No private key configured' });
    return;
  }

  // 4. Call LLM via x402
  const x402Fetch = createX402Client(ogPrivateKey as `0x${string}`);
  const { rawTasks, txHash } = await extractTasksWithProof(x402Fetch, extraction.text);

  // 5. Enrich tasks
  const enrichedTasks: Task[] = rawTasks.map(raw => ({
    ...raw,
    id: generateTaskId(extraction.url, raw.action),
    sourceUrl: extraction.url,
    platform: extraction.platform ?? 'other',
    createdAt: new Date().toISOString(),
    txHash,
    memsyncId: null,
    synced: false,
  }));

  // 6. Save locally
  await saveTasksLocally(enrichedTasks);

  // 7. Async sync to MemSync (don't block response)
  syncToMemSync(enrichedTasks).catch(err => console.warn('[background] MemSync sync failed:', err));

  // 8. Respond to popup
  sendResponse({ success: true, tasks: enrichedTasks, txHash });
}
```
